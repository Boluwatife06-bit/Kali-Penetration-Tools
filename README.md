# Cyber Toolkit

A multi-functional educational cybersecurity tool written in Python.

**Author:** [boluwatife06-bit](https://github.com/boluwatife06-bit)

## Features
- Multithreaded TCP port scanner
- Hash cracker (MD5, SHA1, SHA256) using dictionary attack
- Password strength analyzer & generator
- File encryption/decryption (Fernet)

## Installation
```bash
git clone https://github.com/boluwatife06-bit/CyberToolkit.git
cd CyberToolkit
pip install -r requirements.txt
